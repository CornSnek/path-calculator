Keybind shortcuts:
C and Enter to write and save the cost
S/M/V to mark the node properties above
IJKL or arrow keys (↑↓←→) to select adjacent nodes
Save data as link ?v=... to clipboard.

You can hover over coordinates to highlight it on the grid. Bold numbers represent the running total or sum of previous costs. Previous paths will mark nodes as V.
